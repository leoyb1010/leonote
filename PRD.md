# Leonote PRD（精简版）

## 1. 产品目标
Leonote 是一个面向个人长期使用的中文笔记与轻知识库产品。

目标：
- 记录足够快
- 阅读足够舒服
- 整理足够轻
- 视觉足够克制高级
- 移动端体验足够顺手

## 2. 目标用户
- 用户本人
- 单人长期使用
- 偏重记录、整理、回顾、沉淀

## 3. 核心场景
1. 快速记录想法
2. 浏览最近内容
3. 用标签整理内容
4. 搜索历史笔记
5. 通过每日笔记沉淀日常记录
6. 通过关联笔记形成轻知识库

## 4. 第一版功能范围
### 4.1 必做
- 登录
- 笔记 CRUD
- 自动保存
- 笔记列表页
- 笔记详情 / 编辑页
- 标签系统
- 搜索
- 收藏 / 置顶
- 归档 / 删除
- 每日笔记
- 关联笔记（轻版）
- 图片附件
- 基础设置

### 4.2 暂不做
- 多人协作
- 团队空间
- 端到端加密
- 原生客户端
- AI 功能
- 图谱
- 白板
- 数据库视图
- Web Clipper
- 重同步系统

## 5. 页面结构
- 首页
- 笔记详情页
- 搜索页
- 标签页
- 每日笔记页
- 收藏页
- 归档页
- 设置页

## 6. 交互原则
- 打开即写
- 关键操作不超过 2~3 步
- 移动端优先
- 列表和详情切换清晰
- 输入、保存、返回要稳定

## 7. 视觉原则
- 全中文
- 黑白灰主色
- 单一强调色
- 轻卡片、轻阴影
- 留白适中
- 信息密度高但不压迫

## 8. 成功标准
- 能稳定创建、编辑、搜索、整理笔记
- 手机上顺手可用
- 首页、详情页、搜索页都足够清爽
- 能作为个人长期使用产品上线

## 9. v1.5 记账模块

Leonote 新增轻量记账能力。它不是完整财务管理工具，而是长期记录系统里关于金钱的一条温和信息流。

### 必做
- 5 秒内记一笔开销
- 自定义记账类型：name / emoji / color
- 本月合计、本周合计、类型分布、最近记录
- 首页「今天」增加本周开销轻卡片
- 设置页增加「记账类型」入口

### 暂不做
- 预算 / 告警
- 收入
- 账户 / 支付方式
- CSV 导入导出
- 月度对比 / 趋势预测
- 账单关联笔记
